Food Rescue AI

Project Overview

Food Rescue AI is an AI-powered sustainability project that helps reduce food waste by connecting food donors with nearby NGOs.

Problem Statement

Large amounts of edible food are wasted daily while many people face food insecurity. This project aims to bridge the gap between food donors and NGOs.

Solution

The system uses location, food quantity, NGO capacity, and priority scoring to recommend the best NGO for food collection.

AI Features

- Smart NGO Matching
- Distance Calculation
- Capacity Analysis
- Priority-Based Recommendations

Technologies Used

- Python
- AI-Based Decision Logic
- JSON
- Haversine Distance Formula

Sustainability Impact

- Reduces food waste
- Supports hunger relief
- Promotes sustainable resource utilization
